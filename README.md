# Hey 👋, I'm Saurav

I'm a full-stack developer from India who loves turning ideas into products people actually enjoy using.

I care a lot about clean UI, thoughtful UX, and writing code that's easy to maintain. Whether I'm redesigning an interface, building APIs, or polishing tiny interactions, I enjoy the process of making software feel intuitive.

Some of the things I've worked on:

- Teacher management

- Attendance system

It's the biggest project I've ever worked on, and I'm constantly redesigning, improving, and adding new features.
